如何把你的博客部署到Github Pages上去？


Github账号注册及配置

注册github帐号
建立new repository
如果你没有github帐号，请到官网注册一个，然后去邮箱进行验证；如果你有帐号则直接登录。
官网：https://github.com/ 点击 sign up

注册完后，一般需要验证邮箱，再登录进去
（1）新建一个仓库，仓库名为：jaybinkoo.github.io ，如下图所示

（2）只需填写 username.github.io 即可，然后点击 create repositrory。 注意：username.github.io 的 username 要和 用户名 保持一致，不然后面会失败。以我的为例：

（3）点击github主页右上角的 your profile

（4）找到新建立的 username.github.io 文件,点击打开


（5）点击 settings

（6）往下拉动鼠标到GitHub Pages

（7）点击选择 choose a theme，随便选择一个，（之后我们要更改这些丑陋的模板），然后 select theme 保存就行了.
将Hexo与Github page联系起来，设置Git的user name和email（如果是第一次的话）

（8）配置 Git 个人信息：在 D:\blog\hexo 目录下，鼠标右键打开一个 Git Bash 窗口，输入下面的命令
git config --global user.name " GitHub 用户名 " 然后 回车 。【GitHub 用户名】：就是刚才你注册的用户名 例如我的
git config –global user.email “ GitHub 邮箱 “ 然后 回车。【GitHub 邮箱】：就是刚才你注册的邮箱 例如我的

（9） 生成 SSH KEY，其实就是生成一个公钥和密钥，因为 GitHub 需要一个密钥才能与本地相连接。执行以下命令，并 连续按 3 次回车
生成密钥：| 输入：ssh-keygen -t rsa -C " 邮箱地址 " 例如我的：ssh-keygen -t rsa -C “ jaybin@aliyun.com “

（10）SSH KEY 生成之后,得到了两个文件：id_rsa 和 id_rsa.pub（默认存储路径是：C:\Users\Administrator.ssh）。打开我的电脑，在地址栏输入 C:\Users\Administrator\.ssh ,然后回车，即可打开这个目录 。

（11）鼠标右键选择用记事本打开 id_rsa.pub 文件，复制全部内容，即复制密钥。

（12）打开 GitHub ，依次点击 头像 –>Settings–>SSH and GPG keys –> Add SSH key，将复制的密钥粘贴到 key 输入框，最后点击 Add Key ，SSH KEY 配置成功，如下图所示：


（13）修改 hexo 文件夹下的 _config.yml 全局配置文件，修改 deploy 属性代码，将本地 hexo 项目托管到 GitHub 上，如下图所示： 最好用sublime打开 ， 修改完后， 一定要另存为，编码选择 UTF-8 ,否则会出现乱码

仓库地址可以通过GitHub仓库上，下载按钮获取到：

（14）新建一篇博客，行命令窗口输入：hexo new post “博客名” ，然后 回车. 例如 hexo new post “你好，Hexo”

（15）在生成以及部署文章之前，需要安装一个 hexo-deployer-git 插件扩展
输入 npm install hexo-deployer-git --save，然后 回车 。

（16）执行下面的代码命令： hexo d -g ，将 hexo 项目以及文章托管到 GitHub 上。

注：hexo generate 可缩写为 hexo g ， hexo deploy 可缩写为 hexo d

在这一步，如果是第一次把文件部署到GitHub上，可能会弹出要你输入GitHub账户和密码，你输入一下就行了。
还有一种报错，说发生冲突
解决办法为：

（17）部署成功后,打开浏览器，输入访问你的地址：http://用户名.github.io。那么将看到生成的文章

若担心输错，可以在GitHub pages 上复制下来网址：


到这目前为止你就完成了，将本地hexo框架项目，部署到GitHub上面了，并且可以通过在浏览器里访问你的地址：http://用户名.github.io，查看。
